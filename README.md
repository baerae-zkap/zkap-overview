# ZKAP — Documentation Hub

**English → [en/README.md](./en/README.md)  ·  한국어 → [ko/README.md](./ko/README.md)**

Prove your social login in zero knowledge. Own your account on Ethereum.
ZKAP is open-source public-good infrastructure — a privacy-preserving zk-OAuth +
[ERC-4337](https://eips.ethereum.org/EIPS/eip-4337) account-abstraction protocol.

> ⚠️ **Status:** experimental / testnet. Not audited for production custody of
> real funds.
